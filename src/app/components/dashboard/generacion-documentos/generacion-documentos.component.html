Documentos Dinámicos

Desde ésta sección podrás generar documentos tipo. Deberás agregar el titulo que tendrá el documento y agregar las variables que luego se llenarán para cada trabajador. El cuerpo del documento tendrá cada variable, y luego al ir al trabajador que requieres sea generado el documento; te pedirá rellenar esas variables para generar el documento. También puedes usar variables propias de cada trabajador, están en la lista desplegable de más abajo.
